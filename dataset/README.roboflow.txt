
Grocery store detection - v1 2022-07-05 7:58pm
==============================

This dataset was exported via roboflow.ai on July 5, 2022 at 2:30 PM GMT

It includes 1316 images.
Grocery-items are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Resize to 416x416 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* Random rotation of between -20 and +20 degrees
